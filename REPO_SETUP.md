# GitHub Repository Setup

## Update Repository Settings

1. Go to: https://github.com/tajo9128/Docking-studio/settings

2. **Repository name**: `Biodockify-Studio-AI`

3. **Description**: 
```
Biodockify Studio AI - AI-Powered Autonomous Drug Discovery Platform
```

4. **Website**: Leave blank or add `https://github.com/tajo9128/Docking-studio`

5. **Topics**: Add these topics:
   - molecular-docking
   - drug-discovery
   - ai
   - autodock-vina
   - rdkit
   - openmm
   - molecular-dynamics

## Update Repository Visibility

If you want to make it public:
- Go to: https://github.com/tajo9128/Docking-studio/settings
- Scroll to "Danger Zone"
- Click "Change visibility"
